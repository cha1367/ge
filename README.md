# 대한민국 노동시장 직군 분석 Streamlit 대시보드

## 프로젝트 개요

이 프로젝트는 대한민국 노동시장의 직군 분포 변화와 산업구조 특성을 분석하는 Streamlit 웹 대시보드입니다.  
화이트칼라와 블루칼라를 중심으로 2016년부터 2025년까지의 직군 분포 변화, 관리자 포함/제외 비교, 연령·학력별 특성, 지역별 분포, 임금, 산업별 직군 구조를 시각화합니다.

## 실행 방법

1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

2. Streamlit 실행

```bash
streamlit run app.py
```

## Streamlit Cloud 배포 방법

1. GitHub 저장소를 생성합니다.
2. `app.py`, `requirements.txt`, `README.md`, `data/` 폴더를 업로드합니다.
3. Streamlit Community Cloud에서 해당 GitHub 저장소를 연결합니다.
4. Main file path를 `app.py`로 지정합니다.
5. Deploy를 누릅니다.

## 주의사항

- CSV 파일은 반드시 `data/` 폴더에 넣어야 합니다.
- KOSIS 파일은 첫 행에 단위 행이 포함될 수 있으므로 앱 내부에서 자동으로 제거합니다.
- 구인·미충원 데이터는 직업분류 체계가 다르기 때문에 메인 분석에서 제외했습니다.
- 산업별 직군 구조 분석은 산업별 × 직업별 취업자 데이터를 사용합니다.
