# 쉬었음 청년 사회 확산 대시보드

## 프로젝트 개요

주제: 쉬었음 청년 증가가 노동시장 → 경제 → 재정으로 어떻게 확산되는가

이 대시보드는 20~29세 쉬었음 청년 증가를 단순 원자료로 나열하지 않고, 노동시장·경제·재정 관점의 파생지표로 재구성해 사회적 확산 구조를 보여준다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

## 대시보드 흐름

| 단계 | 화면 | 핵심 내용 |
|---|---|---|
| 1 | 개요 | 분석 문제의식과 전체 흐름 |
| 2 | 문제 규모 | 20~29세 쉬었음 청년 수 추이 |
| 3 | 노동시장 | 쉬었음 증가율, 비경활 중 쉬었음 비율 |

| 5 | 재정 | 20대 고용보험 피보험자 수, 고용보험 기준 사회보험 진입률, 잠재 국민연금 납부 공백 |
| 6 | 정책·결론 | 노동시장·경제·재정 관점의 대응 방향 |
| 7 | 데이터·한계 | 데이터 출처, 수식, 분석 한계 |

## 재정 파트 수정 내용

기존 재정 파트는 국민연금 납부 공백 추정액 중심이어서 소비 공백 차트와 비슷한 모양이 나올 수 있었다. 이를 보완하기 위해 실제 관측자료인 EIS 고용행정통계의 20대 고용보험 피보험자 수를 추가했다.

### 추가 지표

**20대 고용보험 피보험자 수**

```text
EIS 월별 피보험자현황
연령_10세 = 20~29세
지역 = 전체
측정값 = 피보험자수(전체)
각 연도 12월 말 기준
```

**고용보험 기준 사회보험 진입률**

```text
20대 고용보험 피보험자 수 ÷ 20~29세 전체 인구 × 100
```

**잠재 국민연금 납부 공백 추정액**

```text
쉬었음 청년 수 × 271만원 × 12개월 × 9%
```

## 데이터 파일 설명

| 파일 | 설명 |
|---|---|
| resting_youth_20_29.csv | 20~29세 쉬었음 청년 수 |
| inactive_population_20_29.csv | 20~29세 전체 비경제활동인구 |
| youth_population_1age.csv | 1세별 주민등록인구, 20~29세 합산에 사용 |
| employment_insured_20_29.csv | 대시보드용 20대 고용보험 피보험자 연도별 12월 기준 CSV |
| employment_insured_20_29_cleaned.xlsx | 원자료 정제 확인용 엑셀 파일. annual_dec, monthly_clean, metadata 시트 포함 |

## 주의사항

- 고용보험 피보험자수는 월말 시점 인원 수이므로 1월~12월을 합산하면 안 된다.
- 대시보드에서는 각 연도 12월 말 기준 피보험자수만 사용한다.
- 고용보험 기준 사회보험 진입률은 전체 사회보험 가입률이 아니라 고용보험 기준의 보조지표다.
- 소비 공백과 국민연금 납부 공백은 실제 손실액이 아니라 공개 통계와 가정값을 결합한 잠재 추정치다.
