# 청년 취업난 구조 분석 대시보드

KOSIS·고용노동부 공공데이터를 활용해 청년 취업난을 질문 중심으로 분석한 Streamlit 대시보드입니다.

## 핵심 질문

> 청년들은 왜 취업이 어렵다고 느낄까?

본 프로젝트는 청년 취업난을 실업률 하나로 단정하지 않고, 다음 흐름으로 나누어 탐색합니다.

1. 첫 취업까지 오래 걸리는가
2. 취업 준비 상태에 머무르는가
3. 실업률 밖 쉬었음 청년이 있는가
4. 취업해도 안정적인가
5. 임금과 산업별 격차가 있는가
6. 지역별 차이가 있는가

## 이번 보강 반영 사항

- 데이터 출처·기간 표를 대시보드에 명시
- 청년 고용 취약성 지수의 min-max 정규화 산식 명시
- 2015~2019 / 2020~2021 / 2022~2025 시기 구분선을 주요 시계열 차트에 추가
- 취업시험 준비 여부 도넛 차트 제거, 준비자 수·비율 듀얼축 차트로 교체
- 쉬었음 청년 단독 추이 차트를 공식 실업자 수와 쉬었음 청년 수 비교 차트로 교체
- 첫 취업 평균소요기간과 1년 이상 소요 비중을 하나의 듀얼축 차트로 통합
- 정규직·비정규직 비율 2개 라인을 비정규직 비율 중심 차트로 정리
- 주요 지표 간 상관관계 heatmap 추가

## 취약성 지수 산출 방식

청년 고용 취약성 지수는 공식 통계가 아니라 여러 지표의 상대적 흐름을 한눈에 보기 위한 탐색적 요약 지표입니다.

```text
정규화값 = (x - 최솟값) / (최댓값 - 최솟값)

고용률 취약도 = 1 - 정규화(고용률)
첫 취업 지연 취약도 = [정규화(첫 취업 평균소요기간) + 정규화(1년 이상 소요 비중)] / 2
취업시험 준비 취약도 = 정규화(취업시험 준비자 비율)
쉬었음 취약도 = 정규화(쉬었음 비중)
비정규직 취약도 = 정규화(비정규직 비율)
임금 취약도 = 1 - 정규화(월임금총액)

청년 고용 취약성 지수 = 각 취약도 지표의 동일 가중 평균 × 100
```

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포 방법

GitHub 저장소 최상단에 아래 파일과 폴더가 오도록 업로드합니다.

```text
app.py
requirements.txt
README.md
.streamlit/
data/
```

Streamlit Cloud에서는 Main file path를 `app.py`로 지정하면 됩니다.

## 해석상 주의

본 분석은 청년 취업난의 단일 원인을 입증하는 인과분석이 아니라, 관련 지표들의 흐름을 함께 확인하는 탐색적 분석입니다. 쉬었음 청년을 모두 구직 포기자로 해석하거나, 비정규직을 무조건 나쁜 일자리로 단정하지 않습니다.
