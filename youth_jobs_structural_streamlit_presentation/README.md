# 청년 취업난의 구조적 요인 분석

KOSIS·고용노동부 공공데이터를 활용한 Streamlit 대시보드입니다.

## 핵심 방향

이 프로젝트는 청년 취업난을 단순 실업률 문제가 아니라 아래 구조로 분해해 분석합니다.

1. 노동시장 진입 지연
2. 실업률 밖 청년
3. 고용 안정성 부족
4. 취업 후 보상 격차
5. 산업별 일자리 불균형

## 주요 보강 사항

- 발표용 핵심 요약 페이지 추가
- 청년 고용 취약성 지수 민감도 분석 추가
- 최근 취약 영역 기준 정책 우선순위 표 추가
- 데이터 확인 표의 컬럼명 한국어화
- 수치·논리 검토 페이지 포함

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포 방법

GitHub 저장소 최상단에 아래 파일들이 오도록 업로드한 뒤 Streamlit Community Cloud에서 `app.py`를 실행 파일로 지정합니다.

```text
app.py
requirements.txt
README.md
.streamlit/
data/
```

## 해석상 주의

청년 고용 취약성 지수는 공식 통계가 아니라 여러 지표의 상대적 흐름을 보기 위한 탐색적 요약 지표입니다. 인과관계를 확정하는 지표가 아니며, 정책 우선순위 도출을 위한 보조 분석으로 사용해야 합니다.
