# 데이터 수치·논리 검토 보고서

## 결론
현재 데이터는 주제와 대체로 일관성이 있으며, 비현실적으로 튀는 수치는 확인되지 않았다. 다만 일부 지표는 해석을 조심해야 한다.

## 핵심 점검 결과

| 항목 | 점검 결과 | 발표 시 주의 |
|---|---|---|
| 청년 고용률·실업률 | 고용률 41.2~46.6%, 실업률 5.9~9.8%로 현실적 범위 | 2015년 대비와 2022년 이후 흐름을 구분 |
| 비정규직 비율 | 34.6~45.6%로 현실적 범위 | 비정규직=나쁜 일자리 단정 금지 |
| 쉬었음 청년 | 쉬었음 비중 5.3~10.3% | 쉬었음=구직 포기자 단정 금지 |
| 첫 취업 소요기간 | 평균 10~12개월, 1년 이상 비중 26.0~31.3% | 평균과 분포를 함께 제시 |
| 취업시험 준비자 | 12.2~19.1%, 2021년 정점 후 하락 | 계속 증가했다고 표현하지 말 것 |
| 임금 | 명목 월임금 186.9만~269.1만 원 | 임금 하락 주장 금지, 실질임금 한계 명시 |
| 산업별 자료 | 산업별 격차 큼, 일부 소규모 산업 변동 가능 | 보조 분석으로 활용 |
| 취약성 지수 | 2021년 최고, 2025년은 최고점 아님 | 공식 지표가 아닌 탐색적 요약 지표 |

## 안전한 결론 표현
- 청년 취업난은 실업률 하나로 설명하기 어렵다.
- 청년 고용률은 2022년 이후 하락 흐름을 보였다.
- 비정규직 비율, 쉬었음 비중, 첫 취업 장기 소요 비중은 청년 고용의 질과 노동시장 진입 문제를 설명하는 보조 근거가 된다.
- 임금은 명목 기준으로 상승했으므로, 생활 안정성 판단에는 물가와 주거비를 반영한 후속 분석이 필요하다.
- 청년 고용 취약성 지수는 여러 지표의 상대적 흐름을 한눈에 보기 위한 탐색적 요약 지표다.
