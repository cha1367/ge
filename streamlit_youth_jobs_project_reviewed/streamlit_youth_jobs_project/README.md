# 청년 일자리 감소와 취업난 요인 분석 Streamlit 대시보드

## 1. 프로젝트 개요

주제: **청년 일자리 감소와 취업난 요인 분석**  
부제: **고용률·비정규직·비경제활동·임금·산업별 일자리 데이터를 중심으로**

이 대시보드는 KOSIS에서 내려받은 청년 고용 관련 CSV를 전처리하여 Streamlit에서 시각화한 프로젝트입니다.

분석 질문은 다음과 같습니다.

> 청년 취업난은 단순히 일자리 수 부족의 문제인가, 아니면 안정적 일자리 부족·비경제활동 증가·낮은 임금·산업별 일자리 편중과 함께 나타나는 구조적 문제인가?

## 2. 포함 데이터

`data/processed` 폴더에 정리된 CSV가 들어 있습니다.

| 파일명 | 내용 |
|---|---|
| employment.csv | 청년 고용률, 실업률, 취업자, 실업자, 비경제활동인구 |
| nonregular.csv | 청년 임금근로자, 정규직, 비정규직, 비정규직 비율 |
| resting.csv | 15~29세 쉬었음 비경제활동인구 |
| wage.csv | 29세 이하 월임금총액, 월급여액, 시간당임금총액 |
| industry.csv | 산업별 청년 근로자 수와 임금 |

## 3. 로컬 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 4. Streamlit Cloud 배포 방법

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. Streamlit Cloud에서 새 앱을 생성합니다.
3. Repository를 선택합니다.
4. Main file path에 `app.py`를 입력합니다.
5. Deploy 버튼을 누릅니다.

## 5. 보고서 해석 주의

본 프로젝트는 지표 간 흐름과 관련성을 보여주는 데이터 분석 프로젝트입니다.  
특정 지표가 청년 취업난의 직접 원인이라고 단정하지 않고, 관련성·동반 흐름·가능성 중심으로 해석해야 합니다.
