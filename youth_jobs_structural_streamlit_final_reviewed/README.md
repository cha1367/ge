# 청년 취업난의 구조적 요인 분석

KOSIS 및 고용노동부 공공데이터를 활용해 청년 취업난을 단순 고용률·실업률 문제가 아니라 노동시장 진입 지연, 실업률 밖 청년, 고용 안정성, 임금 보상, 산업별 일자리 구조로 분해해 분석하는 Streamlit 대시보드입니다.

## 핵심 구조

- 공식 고용지표: 청년 고용률·실업률·취업자 수
- 노동시장 진입 지연: 첫 취업 소요기간, 취업시험 준비자
- 실업률 밖 청년: 쉬었음 청년, 쉬었음 비중
- 고용 안정성: 정규직·비정규직 비율
- 취업 후 보상: 청년 월임금총액
- 산업 구조: 산업별 청년 근로자 수·월임금
- 청년 고용 취약성 지수: 여러 지표를 표준화한 탐색적 요약 지표

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포

GitHub 저장소 최상단에 `app.py`, `requirements.txt`, `.streamlit/`, `data/`가 오도록 업로드한 뒤 Streamlit Cloud에서 `Main file path`를 `app.py`로 지정합니다.

## 주의

청년 고용 취약성 지수는 공식 통계지표가 아니라 여러 지표의 상대적 흐름을 한눈에 보기 위한 탐색적 요약 지표입니다. 본 프로젝트는 인과관계를 확정하기보다 관련 지표들의 흐름을 비교하는 탐색적 분석입니다.

## 최종 검토 반영 사항
- 데이터 확인 페이지의 표 컬럼명을 한국어로 표시하도록 수정했습니다.
- 수치·논리 검토 페이지를 추가해 비현실적 수치 여부, 해석상 위험한 표현, 발표 방어 포인트를 정리했습니다.
- 서로 다른 단위가 한 그래프에 섞이지 않도록 노동시장 진입 지연 페이지의 그래프를 분리했습니다.
- 청년 고용 취약성 지수 구성요소는 누적 면적그래프가 아니라 선그래프로 표시해 과도한 해석을 줄였습니다.
