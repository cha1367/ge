# 청년 첫 취업 지연 구조 분석 대시보드

## 프로젝트 개요

이 프로젝트는 “청년의 첫 취업은 왜 늦어질까?”라는 질문에서 출발한 Streamlit 대시보드입니다. 청년 취업난 전체의 모든 원인을 입증하기보다는, KOSIS 공공데이터로 확인 가능한 첫 취업 진입 흐름을 정리했습니다.

기존 임금 파트는 제외하고, 새롭게 다음 두 자료를 반영했습니다.

- 성별 취업시험준비유무 및 준비분야
- 성별 첫일자리 산업

## 대시보드 구성

1. 프로젝트 개요
2. 첫 취업까지 얼마나 걸리는가
3. 취업 준비는 어디로 향하는가
4. 실업률 밖 쉬었음 청년
5. 첫 취업 이후 일자리는 안정적인가
6. 첫 취업은 어느 산업으로 연결되는가
7. 지역별 진입 여건은 다른가
8. 종합 결론과 대책

## 주요 수정 사항

- 임금 분석 페이지 삭제
- 취업시험 준비 분야 분석 추가
- 첫 일자리 산업 분포 분석 추가
- 첫 일자리 산업 상위 3개 집중도 파생 지표 추가
- 결론을 정부·기업·대학·지역 관점의 대응 방향으로 수정

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 해석 주의

본 프로젝트는 탐색적 데이터 분석입니다. 첫 취업 지연의 모든 원인을 입증한 것이 아니라, 공개 통계로 확인 가능한 지표의 흐름을 정리한 것입니다.
