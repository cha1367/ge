# 데이터 교차 검토 결과

## 검토 범위

다음 처리 데이터와 대시보드 문구를 교차 검토했다.

- 청년 고용률·실업률·취업자 수
- 첫 취업 소요기간
- 취업시험 준비자
- 쉬었음 청년
- 정규직·비정규직 비율
- 청년 임금
- 산업별 청년 근로자 수·임금
- 시도별 청년 고용률·실업률
- 종합 결론 및 대책 문구

## 수치 검토 결과

심각한 오류나 비현실적 수치는 발견되지 않았다.

| 항목 | 검토 결과 |
|---|---|
| 청년 고용률·실업률 | 고용률 41.2~46.6%, 실업률 5.9~9.8% 범위로 정상적임. 취업자 수 감소는 청년 인구 감소와 함께 해석해야 함. |
| 첫 취업 소요기간 | 평균 10~12개월 범위로 정상적임. 1년 이상 소요 비중도 26.0~31.3% 범위로 해석 가능함. |
| 취업시험 준비자 | 비경제활동인구 중 12.2~19.1% 범위로 정상적임. 2021년 정점 이후 하락 흐름이 있어 “계속 증가”라고 표현하면 안 됨. |
| 쉬었음 청년 | 26.9만~44.8만 명, 비경제활동인구 중 5.3~10.3% 범위로 정상적임. 구직 포기자로 단정하지 않도록 문구 유지. |
| 비정규직 비율 | 34.6~45.6% 범위로 정상적임. 정규직과 비정규직 합계가 일부 연도에서 반올림 때문에 임금근로자 합계와 1천 명 정도 차이가 날 수 있음. |
| 청년 임금 | 월임금총액 186.9만~269.1만 원 범위로 정상적임. 명목임금이므로 실질임금처럼 해석하지 않도록 주의 문구 유지. |
| 산업별 자료 | 근로자 수와 월임금 범위가 대체로 정상적임. 소규모 산업은 변동성이 클 수 있어 사분면 분석의 보조자료로 해석하는 것이 안전함. |
| 지역별 고용률·실업률 | 시도별 고용률 31.5~51.8%, 실업률 3.1~12.6% 범위로 정상적임. 지역별 실업률은 변동성이 클 수 있어 보조지표로 처리함. |
| 청년 고용 취약성 지수 | 32.8~56.8 범위로 정상 계산됨. 공식 지표가 아닌 탐색적 요약 지표라는 설명을 유지해야 함. |

## 수정한 부분

1. 지역별 분석 페이지를 추가하면서 프로젝트 개요의 분석 흐름이 기존 “다섯 가지 질문”으로 남아 있던 부분을 수정했다.
   - 수정 전: 청년 취업난을 다섯 가지 질문으로 나누어 살펴본다.
   - 수정 후: 청년 취업난을 질문형 흐름으로 나누어 살펴본다.

2. 개요 카드에 지역별 분석을 반영했다.
   - 추가 항목: 지역별 차이가 있는가
   - 설명: 시도별 청년 고용률과 실업률로 전국 평균 뒤의 지역별 차이를 보조적으로 확인한다.

## 해석상 주의할 점

- 이 프로젝트는 청년 취업난의 인과관계를 입증하는 분석이 아니라, 관련 지표의 흐름을 확인하는 탐색적 분석이다.
- 쉬었음 청년은 모두 구직 포기자로 해석할 수 없다.
- 비정규직은 항상 나쁜 일자리라고 단정할 수 없다.
- 임금 자료는 명목임금이므로 물가와 주거비를 반영한 실질 생활 수준으로 직접 해석하면 안 된다.
- 지역별 고용률만으로 특정 지역의 취업난을 단정하면 안 된다.
- 취약성 지수는 공식 통계가 아니라 여러 지표를 요약한 보조 지표다.

## 최종 판단

현재 대시보드의 데이터, 그래프, 결론 문구는 주제와 대체로 일관적이다. 교수님이나 대중이 보았을 때 가장 위험한 단정 표현은 제거되어 있으며, 각 데이터는 다음 논리 흐름에 맞게 연결되어 있다.

1. 첫 취업까지 오래 걸리는가
2. 취업 준비 상태에 머무르는가
3. 실업률 밖 쉬었음 청년이 있는가
4. 취업해도 안정적인가
5. 임금과 산업별 격차가 있는가
6. 지역별 차이가 있는가

따라서 현재 버전은 수치상 큰 오류 없이 발표용으로 사용할 수 있다.
