# 쉬었음 청년 사회 확산 대시보드

## 주제

**쉬었음 청년 증가가 노동시장 → 경제 → 재정으로 어떻게 확산되는가**

이 대시보드는 2015~2025년 20~29세 쉬었음 청년 증가를 노동시장, 경제, 재정 관점의 파생지표로 재구성해 보여준다.

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

Streamlit Cloud에서는 Main file path를 다음으로 설정한다.

```text
dashboard.py
```

## 파일 구성

| 파일 | 설명 |
|---|---|
| dashboard.py | Streamlit 대시보드 코드 |
| requirements.txt | 실행 패키지 목록 |
| resting_youth_20_29.csv | 20~29세 쉬었음 청년 수 |
| inactive_population_20_29.csv | 20~29세 전체 비경제활동인구 |
| youth_population_1age.csv | 20세~29세 주민등록인구 합산용 자료 |
| employment_insured_20_29.csv | 20대 고용보험 피보험자 수, 각 연도 12월 말 기준 |
| employment_insured_20_29_cleaned.xlsx | 고용보험 피보험자 자료 정제본 |

## 대시보드 흐름

1. 개요
2. 문제 규모
3. 노동시장
4. 경제
5. 재정
6. 정책·결론

## 정책·결론 핵심

최종 정책 방향은 **우수 중소·중견기업 실무경력 인증 우대전형**이다.

쉬었음 청년 대책은 단순한 지원금 확대가 아니라, 청년이 실제 일자리로 진입하고 유지될 수 있는 경력 사다리를 설계하는 방향이어야 한다. 우수 중소·중견기업에서 쌓은 실무경력을 공식 인증하고, 이를 대기업·상위 중견기업·공공 기술직 채용에서 우대하는 협약형 전형으로 연결한다.

## 주의
