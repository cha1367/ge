# 청년 취업 흐름 분석 대시보드

KOSIS·고용노동부 공개통계로 청년 취업 상황을 질문별로 정리한 Streamlit 대시보드입니다.
고용률 하나만 보지 않고, 첫 취업까지 걸리는 시간, 취업 준비 상태, 쉬었음 청년, 비정규직, 임금·산업 구조, 지역별 차이를 함께 봅니다.

## 대시보드 질문

1. 첫 취업까지 시간이 오래 걸리는가?
2. 취업 준비 상태에 오래 머무는가?
3. 실업률 밖에 쉬었음 청년이 있는가?
4. 취업해도 일자리는 안정적인가?
5. 어떤 산업에 취업하고, 얼마나 받는가?
6. 지역마다 청년 고용 상황이 다를까?

## 사용 데이터

- 청년 고용률·실업률·취업자 수
- 첫 취업 소요기간
- 취업시험 준비자
- 쉬었음 청년
- 정규직·비정규직 비율
- 청년 평균 임금
- 산업별 청년 근로자 수·임금
- 시도별 청년 고용률·실업률

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포

Streamlit Community Cloud에서 `app.py`를 main file path로 지정하면 됩니다.

## 해석 기준

이 대시보드는 청년 취업 문제의 원인을 단정하지 않습니다. 공개 통계로 확인되는 흐름을 정리하고, 각 지표가 어떤 부분을 보여주는지 설명하는 데 초점을 둡니다. 지역별 고용률·실업률은 전국 평균을 보완하는 참고 자료로 사용했습니다.

## 최근 수정 내용

- 청년 인구·취업자 수·고용률을 2015년=100 지수로 함께 비교했습니다.
- 산업명 표준화로 과거/최근 산업분류 명칭 차이를 정리했습니다.
- 시간당임금 2020년 변동, 세종 실업률 결측, 반올림 오차에 대한 설명을 추가했습니다.
- 결론 페이지에서 복잡한 지수 파트를 제거하고, 결론과 대응 방향 중심으로 정리했습니다.
- 대시보드 문구를 발표·보고서에 쓰기 쉬운 자연스러운 문장으로 다듬었습니다.


## 디자인

- 다크 프리미엄 분석 대시보드 스타일을 적용했습니다.
- 데이터, 분석 흐름, 결론 구조는 기존 최종본을 유지했습니다.
