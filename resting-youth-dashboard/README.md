# 쉬었음 청년 사회 확산 대시보드

## 프로젝트 개요

**주제:** 쉬었음 청년 증가가 노동시장 → 경제 → 재정으로 어떻게 확산되는가

단순히 쉬었음 청년 수가 늘었다는 사실을 넘어,  
그 증가가 사회 전체에 어떤 파급 효과를 가져오는지 파생지표로 시각화한 대시보드입니다.

---

## 대시보드 흐름

| 단계 | 관점 | 핵심 지표 |
|---|---|---|
| STEP 1 | 문제 규모 확인 | 쉬었음 청년 수 추이 |
| STEP 2 | 노동시장 | 청년 이탈 가속도 지수 / 비경활 중 쉬었음 비율 |
| STEP 3 | 경제 | 청년 소비 공백 추정액 / 청년 내수 기여 공백률 |
| STEP 4 | 재정 | 잠재 국민연금 납부 손실 / 누적 손실 추정 |
| STEP 5 | 정책 시사점 | 대책방안 3가지 |

---

## 파생지표 수식

### 경제 관점
**청년 소비 공백 추정액**
```
쉬었음 청년 수 × 271만원 × 0.69
```

**청년 내수 기여 공백률**
```
쉬었음 청년 수 ÷ 20~29세 전체 인구 × 100
```

### 재정 관점
**잠재 국민연금 납부 손실 추정액**
```
쉬었음 청년 수 × 271만원 × 9%
```

## 데이터 출처

| 파일 | 출처 | 기간 |
|---|---|---|
| resting_youth_20_29.csv | KOSIS 경제활동인구조사 | 2015~2025 |
| inactive_population_20_29.csv | KOSIS 경제활동인구조사 | 2015~2025 |
| youth_population_1age.csv | KOSIS 주민등록인구통계 | 2015~2025 |

**고정값**
- 20대 평균소득 271만원: 통계청 2024년 임금근로일자리 소득결과
- 평균소비성향 0.69: 통계청 가계동향조사
- 국민연금 보험료율 9%: 국민연금법 제88조
