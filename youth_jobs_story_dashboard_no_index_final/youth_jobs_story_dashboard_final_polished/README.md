# 청년 취업난의 구조적 요인 분석

KOSIS·고용노동부 공공데이터를 활용해 청년 취업난을 질문 중심으로 분석한 Streamlit 대시보드입니다.

## 분석 질문

1. 청년들이 첫 취업까지 오래 걸리는가?
2. 취업 준비 상태에 오래 머무르는가?
3. 실업률에 잡히지 않는 쉬었음 청년이 있는가?
4. 취업해도 비정규직 비율이 높은가?
5. 취업 후 임금과 산업별 격차가 있는가?
6. 청년 취업난은 지역마다 다르게 나타나는가?

## 주요 데이터

- 청년 고용률·실업률·취업자 수
- 첫 취업 소요기간
- 취업시험 준비자
- 쉬었음 청년
- 정규직·비정규직 비율
- 청년 평균 임금
- 산업별 청년 근로자 수·임금
- 시도별 청년 고용률·실업률

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포

Streamlit Community Cloud에서 `app.py`를 main file path로 지정하면 됩니다.

## 해석 주의

본 프로젝트는 인과관계를 입증하는 분석이 아니라 청년 취업난과 관련된 여러 지표의 흐름을 확인하는 탐색적 분석입니다. 지역별 청년 고용률·실업률은 전국 단위 분석을 보완하는 참고 분석으로 사용했습니다.
